## 提交前准备

### OpenReview 账号
- [ ] 所有作者都有 OpenReview 账号
- [ ] 所有作者 profile 完整（affiliation, DBLP 链接等）
- [ ] 注册 NeurIPS 2026 Conference

### 提交材料清单
- [ ] `main.pdf` — 主论文 (≤10 页 + references + appendix)
- [ ] `supplementary.zip` — 补充材料
  - `code/` — 核心代码（anonymized）
  - `full_results/` — 完整实验结果
  - `appendix.pdf` — 如果 appendix 单独提交
- [ ] Author list + affiliations（匿名模式下不显示）
- [ ] Keywords / topics 选择
- [ ] Abstract (≤250 words)
- [ ] NeurIPS Paper Checklist（在线填写）

### Paper Checklist 关键项
1. Claims: 是否有明确的 contribution statement? ✅
2. Limitations: 是否讨论了局限性? ✅
3. Theory: N/A (无理论证明)
4. Experiments: 是否报告了 error bars? ✅
5. Experiments: 是否说明了计算资源? ✅
6. Code: 是否承诺公开? (建议: "Code will be released upon acceptance")
7. Data: 是否使用了公开数据集? ✅
8. Ethics: 是否有 broader impact? ✅

## 应急方案

### 如果 5/04 前论文没写完
- 先提交 abstract（仅需 title + abstract + keywords）
- 5/06 前赶完全文

### 如果某个实验来不及
- 在论文中标注 "results forthcoming in camera-ready"
- 把已有结果先提交
- rebuttal 时补充

### 如果 OpenReview 出问题
- 提前 24h 测试提交流程
- 保留本地备份
- 找同事代提交（最坏情况）
