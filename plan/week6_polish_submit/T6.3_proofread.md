# T6.3 校对 + 匿名化

## 基本信息
- **优先级**: P0
- **预计耗时**: 2 天
- **产出**: 终稿 PDF
- **状态**: [ ] 未开始

## 校对 Checklist

### 内容
- [ ] 所有 claim 都有实验数据支撑
- [ ] 数字和表格/图之间没有矛盾
- [ ] 引用完整（每个提到的方法都有 citation）
- [ ] 没有 "we will show" / "we plan to" 等未来式（改为完成式）
- [ ] Contribution 与实验结果匹配

### 格式
- [ ] 主文 ≤ 10 页（不含 references 和 appendix）
- [ ] NeurIPS 2026 style file (`neurips_2026.sty`)
- [ ] References 格式统一
- [ ] Figure/Table 编号和引用一致
- [ ] 没有 orphan 段落（单行开头/结尾）

### 语言
- [ ] 没有语法错误
- [ ] 一致的时态（实验描述用过去式，方法描述用现在式）
- [ ] 缩写首次出现时展开 (RL, MDP, GRPO, etc.)
- [ ] 没有过于口语化的表达

### 数学
- [ ] 符号全文一致（检查 MASTER_PLAN 中的记号表）
- [ ] 公式编号和引用正确
- [ ] 下标/上标清晰

## 匿名化 Checklist

- [ ] 删除所有作者姓名和单位
- [ ] 删除 GitHub / HuggingFace 链接
- [ ] 删除 "our lab" / "our university" 等表述
- [ ] 检查 PDF metadata（作者字段清空）
- [ ] 检查图片中是否有身份信息
- [ ] 检查 Acknowledgements（提交时删除，接收后补回）
- [ ] 使用 `\usepackage[anonymous]{neurips_2026}` 或等效匿名选项
- [ ] 自引用使用第三人称（"X et al. (2025)" 而非 "our previous work"）

## 常见被拒原因（最后检查）

- [ ] Claims 太强但证据不足？ → 加 hedge 或补实验
- [ ] 与 Conductor 的对比是否公平？ → 承认其优势，强调自身差异
- [ ] Error bars 是否到位？ → 所有主要数字需要 std
- [ ] Reproducibility 信息是否充分？ → 超参数表、代码说明
- [ ] Broader impact 是否有？ → NeurIPS 要求

## 验收标准
- [ ] 全文无语法/拼写错误
- [ ] 匿名化完成
- [ ] PDF 编译无 warning
- [ ] 主文 ≤ 10 页
- [ ] 打印预览检查（图表清晰度）
