### P1: 应该补

**3. β 和 γ 的敏感性**
- W3 只做了 α 的扫描
- Reviewer 可能问 β (turn penalty) 和 γ (efficiency bonus) 的影响
- 至少补 β = {0, 0.1, 0.2} 和 γ = {0, 0.15, 0.3}

**4. 不同 worker LLM 的影响**
- 将 executor_strong 从 Claude 换成 GPT-4o 或 Gemini-Pro

### P2: 可选

**5. 训练数据量的影响**
- 用 1K / 3K / 5K / 7K 训练数据，观察效果变化
- 回答 "需要多少训练数据?"

**6. 不同 base model**
- 将 Qwen2.5-7B 换成 Llama-3.2-7B
- 证明方法的 base model 无关性
