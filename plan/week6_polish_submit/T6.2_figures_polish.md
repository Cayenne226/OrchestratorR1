## 图表质量标准

### 通用要求
- **DPI**: ≥ 300 (PDF 矢量优先)
- **字体**: 与 NeurIPS template 一致 (Times New Roman 或 Computer Modern)
- **字号**: 轴标签 ≥ 10pt，标题 ≥ 11pt，图例 ≥ 9pt
- **配色**: 色盲友好 (避免红绿直接对比)，推荐 Colorbrewer 调色板
- **线宽**: ≥ 1.5pt
- **Marker**: 不同方法用不同 marker (●, ★, ▲, ■)
- **去掉 matplotlib 默认灰色背景**
- **去掉多余 spines (top, right)**

COLORS = {

```python
import matplotlib

```
