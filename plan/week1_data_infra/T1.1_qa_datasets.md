## 统一格式

```jsonl
{"input": "What is the capital of France?", "answer": "Paris", "source": "nq", "difficulty": "simple"}
{"input": "Who directed the film...", "answer": ["answer1", "answer2"], "source": "hotpotqa", "difficulty": "multi_hop"}
```

字段说明:
- `input`: 问题文本
- `answer`: 标准答案（字符串或列表）
- `source`: 数据集来源标识
- `difficulty`: `simple` | `multi_hop`（用于后续分组分析）

## 实现步骤

1. **修改 `data_process/prepare_data.py`**:
   - 添加 TriviaQA, PopQA, 2WikiMultihopQA, MuSiQue 的加载逻辑
   - 添加 `difficulty` 字段标注
   - 添加 `--max_per_source` 参数控制每个数据集抽样量
   - 添加 `--split train/test` 参数

2. **数据清洗**:
   - 去除空答案
   - 答案标准化（小写、去标点）
   - 去重（按 question 文本）
   - 长度过滤（问题 < 512 tokens）

3. **生成训练/测试集**:
   ```bash
   python data_process/prepare_data.py \
       --sources nq,triviaqa,popqa,hotpotqa,2wiki,musique \
       --max_per_source 1000 --split train \
       --output data/train_qa.jsonl

   python data_process/prepare_data.py \
       --sources nq,triviaqa,popqa,hotpotqa,2wiki,musique \
       --max_per_source 500 --split test \
       --output data/test_qa.jsonl
   ```

4. **验证**:
   - 每个 source 的样本数正确
   - 无空答案
   - format 可被 train.py 正确读取

## 完成备注
- PopQA 仅有 test split，训练时从 test split 中取样（通过 seed=42 + shuffle 区分训练/测试）
- 2WikiMultihopQA 使用 `ohjoonhee/2WikiMultihopQA` 替代原 `xanhho/` 版本（原版 dataset script 已弃用）
- MuSiQue 使用 `bdsaglam/musique` 替代原 `drt/musique`（原版不存在）
- difficulty 字段: simple (NQ, TriviaQA, PopQA), multi_hop (HotpotQA, 2Wiki, MuSiQue)
