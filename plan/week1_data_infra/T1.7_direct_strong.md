# T1.7 Direct-GPT-4o Baseline

## 基本信息
- **优先级**: P1
- **预计耗时**: 0.5 天
- **依赖**: T1.1-T1.3 (数据)
- **产出**: `eval/results/direct_gpt4o.json`
- **预计 API 成本**: ~$15-20
- **状态**: [ ] 未开始

## 目标
评估直接调用 GPT-4o 回答问题（无编排、无 agent）的表现，作为"不编排的强模型"上限参考。

## 为什么需要这个
- 建立"花多少钱能买到多好的结果"参考线
- 如果 OrchestratorR1 用更低成本接近 Direct-GPT-4o → 证明编排的成本效率
- 如果 OrchestratorR1 超过 Direct-GPT-4o → 证明多 agent 协作的质量优势
