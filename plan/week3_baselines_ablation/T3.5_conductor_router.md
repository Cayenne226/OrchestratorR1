## Matched-Worker 实验设计

### Agent Pool 对齐方案

将我们的 6 个功能 Agent 映射到 Conductor 级别的 worker：

| Agent Type | Cheap Config | Matched Config |
|------------|-------------|----------------|
| executor | GPT-4o-mini | GPT-5 |
| refiner | GPT-4o-mini | Claude Sonnet 4 |
| decomposer | GPT-4o-mini | Gemini 2.5 Pro |
| critic | GPT-4o | DeepSeek-R1-32B |
| synthesizer | GPT-4o-mini | Qwen3-32B |
| knowledge | GPT-4o-mini | Gemma3-27B |

> **注意**: 具体映射可调整，关键是 matched 配置的 worker 能力等级要与 Conductor 对齐。

### 实现方式

```python
# route_service.py 中添加 worker pool 配置切换
WORKER_POOLS = {
    "cheap": {
        "executor": {"model": "gpt-4o-mini", "cost_per_1M": 0.15},
        "refiner": {"model": "gpt-4o-mini", "cost_per_1M": 0.15},
        ...
    },
    "matched": {
        "executor": {"model": "gpt-5", "cost_per_1M": 10.0},
        "refiner": {"model": "claude-sonnet-4", "cost_per_1M": 3.0},
        ...
    }
}
```

### 评估命令
```bash
# Matched-worker: GPQA + LiveCodeBench
python eval/eval_orchestrator.py \
    --model_path checkpoints/orch_grpo_seed1/final \
    --data_path data/test_gpqa.jsonl data/test_livecode.jsonl \
    --worker_pool matched \
    --output eval/results/ours_matched_gpqa_lcb.json

# Cheap-worker: 全部 10 个 benchmark
python eval/eval_orchestrator.py \
    --model_path checkpoints/orch_grpo_seed1/final \
    --data_path data/test_qa.jsonl data/test_code.jsonl data/test_gpqa.jsonl data/test_livecode.jsonl \

```
