## 预期涌现模式

| 模式 | 含义 | 证据 |
|------|------|------|
| 简单 QA → exec_cheap 为主 | 简单题走捷径 | ✅ 自适应 |
| 多跳 → decomposer + synthesizer 高频 | 复杂题自动分解+综合 | ✅ 涌现分解行为 |
| 代码 → exec_strong + critic 高频 | 代码题需要强推理+验证 | ✅ 任务感知 |
| 简单 QA 几乎不用 critic | 不浪费成本在简单题上 | ✅ 成本感知 |

## 实现

```python
# analysis/agent_distribution.py
import matplotlib.pyplot as plt
import seaborn as sns
import json
import numpy as np

def plot_heatmap(eval_json_path, output_path):
    data = json.load(open(eval_json_path))

    sources = ["nq", "triviaqa", "popqa", "hotpotqa", "2wiki", "musique", "humaneval", "mbpp"]
    agents = ["refiner", "decomposer", "executor_cheap", "executor_strong", "critic", "synthesizer"]

    # 构建矩阵
    matrix = np.zeros((len(sources), len(agents)))
    for i, src in enumerate(sources):
        src_results = [r for r in data["results"] if r["source"] == src]
        total_calls = sum(len(r["agent_calls"]) for r in src_results)
        for j, agent in enumerate(agents):
            count = sum(1 for r in src_results for c in r["agent_calls"] if c["agent_type"] == agent)
            matrix[i][j] = count / max(total_calls, 1)

    # 绘制
    fig, ax = plt.subplots(figsize=(10, 6))
    sns.heatmap(matrix, annot=True, fmt=".2f", cmap="YlOrRd",
                xticklabels=agents, yticklabels=sources, ax=ax)
    ax.set_xlabel("Agent Type")
    ax.set_ylabel("Dataset")
    ax.set_title("Agent Call Distribution (Normalized per Dataset)")
    plt.tight_layout()
    plt.savefig(output_path, dpi=300, bbox_inches='tight')
```

## 论文叙述

> Figure X shows the agent call distribution across datasets.
> Simple factoid questions (NQ, TriviaQA) predominantly invoke `executor_cheap` in a single step,
> while multi-hop reasoning tasks (HotpotQA, MuSiQue) trigger `decomposer` and `synthesizer` at high rates.
> Code tasks (HumanEval) show heavy use of `executor_strong` and `critic`.
> **These patterns emerge purely from RL training with a single reward function,
> without any explicit task-type conditioning or hand-designed routing rules.**
