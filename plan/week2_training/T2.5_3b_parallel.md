# T2.5 3B 对比训练

## 基本信息
- **优先级**: P1
- **预计耗时**: SFT ~15min + GRPO ~4-6h
- **依赖**: W1 全部
- **产出**: `checkpoints/sft_warmup_3b/`, `checkpoints/orch_grpo_3b_seed1/final/`
- **状态**: [ ] 未开始

## 目标
训练 Qwen2.5-3B-Instruct 版本，用于：
1. Scaling analysis (3B vs 7B)
2. 与 Router-R1 直接对比（Router-R1 也是 3B）
3. 训练调试（3B 更快，可先验证 pipeline）

## 执行策略

### 如果有空余 GPU（另一台机器或部分 GPU）:
- 与 7B 训练并行
- 3B 只需 2 张 A100 即可

### 如果只有一台 4×A100:
- 在 7B Seed 1 和 Seed 2 之间的空隙运行
- 或在 7B 评估时（仅需 1 GPU）运行

## 训练配置差异

| 参数 | 7B | 3B |
|------|-----|-----|
| model | Qwen2.5-7B-Instruct | Qwen2.5-3B-Instruct |
| G | 16 | 16 |
| batch_size | 4 | 8 (3B 显存更少) |
| grad_accum | 4 | 2 |
| max_prompt_length | 8192 | 8192 |
| lr | 1e-6 | 1e-6 |
| total_steps | ~2000 | ~2000 |

## 命令

```bash
# SFT
accelerate launch --config_file training/accelerate_fsdp_a100_4gpu.yaml \
    training/sft_warmup.py \
    --model_path models/Qwen2.5-3B-Instruct \
    --data_path data/sft_warmup.jsonl \
    --output_dir checkpoints/sft_warmup_3b

# GRPO
bash training/train_a100.sh \
    --model_path checkpoints/sft_warmup_3b \
    --output_dir checkpoints/orch_grpo_3b_seed1 \
    --seed 42
```
