## 结构 (~2.5 页)

### 3.1 Problem Formulation (~0.5 页)
- MDP 四元组 (S, A, T, R) 定义
- 状态 s_t = (x, a_1, o_1, ..., a_{t-1}, o_{t-1})
- 动作 a_t ∈ {(agent_type, query)} ∪ {TERMINATE(answer)}
- 转移 T: 确定性，由 agent API 返回
- 奖励 R(τ): 终端稀疏奖励

### 3.2 Orchestrator Policy (~0.3 页)
- π_θ(a_t | s_t)，基座 Qwen2.5-7B-Instruct
- 训练目标 J(θ) = E[R(τ)]
- GRPO 梯度估计

### 3.3 Agent Pool (~0.4 页)
- 6 种功能角色表格（角色、功能、底层模型、成本）
- 固定不训练
- 与 Router-R1 (同质模型选择) 的区别

### 3.4 Reactive Generation Loop (~0.5 页)
- Algorithm 1 伪代码
- 与 Conductor (开环) 的区别
- State masking: 仅优化 orchestrator 生成的 token

### 3.5 Reward Function (~0.4 页)
- R(τ) = R_outcome − α·C_cost − β·C_turns + γ·B_eff + R_format
- 各项定义和直觉
- R_format = -1.0 的设计动机

### 3.6 Training (~0.4 页)
- Stage 0: SFT warmup (200 examples, format learning)
- Stage 1: GRPO (G=16, α=0.3, β=0.1, γ=0.15)
- 训练阶段 API 成本控制策略
